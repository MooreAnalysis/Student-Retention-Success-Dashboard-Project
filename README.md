# Student-Retention-Success-Dashboard-Project
This project analyzes student retention, completion, GPA outcomes, and student success indicators using a simulated higher education dataset. The dashboard was developed in Power BI to support institutional effectiveness and data-informed student success initiatives.

# Dataset
- Simulated higher education dataset

# Key Insights
- Computer Science students demonstrated the lowest completion rate at 86% compared to other academic programs included in the analysis.
- Computer Science students who participated in Peer Mentoring achieved an average GPA of 3.0, compared to 2.5 among students who did not participate.
- This also validates the effectiveness of peer mentoring program framework
- Findings supported recommendations for proactive advising outreach, expanded peer mentoring opportunities, and targeted academic support initiatives.

# New Challenges
- Cleaning and standardizing the dataset to ensure fields such as retention status, completion indicators, and GPA values were formatted consistently for analysis in Power BI.
- Determining which metrics and visualizations most effectively communicated student success trends without overcrowding the dashboard.
- Designing an interactive dashboard layout that balanced usability, readability, and data storytelling for a higher education audience.

# Data Preparation
- Imported and cleaned a simulated higher education dataset in Power BI Desktop.
- Fixed inconsistent program names, enrollment formats, and Yes/No values.
- Removed duplicate records and handled missing GPA, tutoring, and financial aid data.
- Standardized data types and formatting for accurate analysis and reporting.
- Created DAX measures and calculated columns for retention, completion, graduation, and GPA metrics.
- Prepared the dataset for interactive dashboards, filtering, and student success analysis.

# Analytical Techniques
- Descriptive analysis to evaulate student GPAs, completion rates
- Comparative analysis to identify differences in academic outcomes between peer mentoring participants and non-participants and first generation status
- KPI reporting to monitor student success
- Data segmentation and interactive filtering to analyze average student GPA by program, peer mnetoring participation, and demmographic factors such as first generation status
